## What Simply Starter Is

A free WordPress theme that provides structure and a shared token system.
It is intentionally brandless — clean layout, no colors baked in.
Brand, fonts, and advanced layout building come from the Simply Suite (see below).

## What You Can Do Right Now (Free)

**Page structure:**
- Fixed header, homepage hero overlap, interior padding
- Page Builder template (full-width, no title/meta)
- Standard Post template (800px centered, featured image hero)
- Landing page template (no nav/footer — for evites, campaigns, event pages)
- 4-column footer widget area

**Section colors** — add a class to any container:
- `.is-dark` `.is-light` `.is-brand-1` `.is-brand-2`

**Style tokens** — these exist across every Simply plugin and theme element.
Set them once and everything updates:
```css
--client-accent        /* buttons, badges, highlights */
--client-heading       /* heading color */
--client-font-display  /* display/title font */
--client-font-primary  /* body font */
--client-radius        /* corner radius — cards + buttons */
--client-nav-bg        /* navigation background */
```

**Free plugins that work with Simply Starter:**
Simply News, Simply Events, Simply Team, Simply Evite, Simply FAQs,
Simply Logo Slider, Simply Utility Bar — all read the tokens above automatically.
Each has its own Simply AI Guide with plugin-specific instructions.
